HexSpire
Color Selector

Color inspiration tool

Description

Using the Hexbot API, HexSpire displays a selection of random colors. The user can view a color's hex value, add colors to the favorite section, and copy the hex color codes from the favorites to the clipboard for additional use. The color grid can be sorted by hue and refreshed to display a new set of random colors.

Features

  • A single-page application, built using HTML, CSS, vanilla JavaScript, and the Hexbot API.
  • To sort by hue, the hex color codes are converted to the HSB color space.
  • A responsive design that displays more colors as the screen size increases.
  • The text color (black or white) is determined automatically based on the color's YIQ ratio.
  • Hex color codes are copied to the clipboard using the asynchronous Clipboard API.

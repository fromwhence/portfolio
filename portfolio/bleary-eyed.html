Bleary-eyed
Celebrity Quiz

Photo identification

Description

The Bleary-Eyed Quiz is a celebrity quiz that presents users blurry photos to identify for points. With each incorrect answer the image becomes less blurry and the potential points decrease.

Features

  • A single-page application, built using HTML, CSS, and vanilla JavaScript.
  • Celebrity order and multiple choice names are displayed at random.
  • Responsive design that utilizes CSS flexbox and media queries to properly display quiz across different devices.
  • The blurred effect is acheived using the CSS filter property and JavaScript.
  • The social media sharing feature in final score modal utilizes AddThis.

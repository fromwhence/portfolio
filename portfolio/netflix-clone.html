Netflix
Clone

Scrimba final project

Description

Built using React, the Netflix Clone displays a selection of random colors. The user can view a color's hex value, add colors to the favorite section, and copy the hex color codes from the favorites to the clipboard for additional use. The color grid can be sorted by hue and refreshed to display a new set of random colors.

Features

  • A React application, built using JSX, JavaScript, CSS, and Firebase.
  • Data for the movies and TV series is stored and accessed using Cloud Firestore.
  • React compound components are used to build the component library.
  • React styled components are used to apply CSS into reusuable components.
  • User sign-in/authentification is managed with Firebase Authentification.

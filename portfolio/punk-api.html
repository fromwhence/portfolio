Punk API
Beer Filter

Filter by ABV & IBU

Description

The Punk API Beer Filter began as a project from the Scrimba Frontend Development Career Path to learn the Fetch API. The application provides the user the ability to filter the Punk API's beer list by alcohol volume (ABV) and/or hoppiness (IBU) and view detailed information about the individual beers.

Features

  • A single-page application, built using HTML, CSS, vanilla JavaScript, and the Punk API.
  • Modifiable SVG icons are used to represent levels of hoppiness and alcohol volume.
  • Using JavaScript, results can be sorted by beer name, alcohol volume, and hoppiness.
  • On larger devices, the results can be viewed in either a grid or list view.
  • Responsive design that utilizes CSS grid, flexbox, and media queries to properly display the application across different devices.
  • Using JavsScript, a dropdown menu allows the user to select their preferred results quantity per page.
